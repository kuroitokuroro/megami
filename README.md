こちらは、開発を補助するためのツールです。

タイトル：開発サポートめがみ<br>
ジャンル：開発日記、兼ファイルリネーム<br>
推奨環境：PC　Chrome<br>
進捗：もう使える　自分で使いながら改良中<br>

大きな開発に取り組むとき、「〇〇を実装する直前のファイルを、それと明確にわかる状態で残しておきたい」「まるでゲーム世界のセーブポイントみたいに、あのときのデータが欲しい」と感じるようになりました。そんな私の願いを叶えるためのツールです。

開発日記に「削除機能」と入力すると、「削除機能　の開発前」の形式で日記をつけます。

さらに、アップロードしたファイルをコピーしてファイル名に日付と時刻を入れ、「index.html」→「0611_0749_削除機能_の開発前_index.html」の形式にし、リネームしたファイルを指定されたファイルにダウンロードします。

リネーム時に西暦を入れないのは、「一年以内にはファイルを整理整頓できる人間でありたい」という私個人の怠惰な願いによるものです。
